# Contributor Covenant Code of Conduct

## 我们的承诺

为了营造一个开放友好的环境，作为贡献者和维护者，我们承诺让参与 CalNexus 项目及社区的每个人都不受骚扰，无论年龄、体型、可见或不可见的残疾、族裔、性别认同与表达、经验水平、教育、社会经济地位、国籍、个人外貌、种族、宗教或性取向身份。

我们承诺以促进开放、包容、多元化、健康社区的方式行动和互动。

## 我们的准则

创造积极环境的行为示例包括但不限于：

- 对他人表现出同理心和善意
- 尊重不同的主张、观点和感受
- 给予和优雅地接受建设性反馈
- 承担责任并向受我们错误影响的人道歉，并从中学习
- 关注的不仅是个人，更是整个社区的福祉

不可接受的行为示例包括但不限于：

- 使用性化语言或意象，以及任何形式的性骚扰或挑逗行为
- 钓鱼、侮辱或贬损的评论，以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人的私人信息，如物理地址或电子邮箱
- 其他可被合理认定为不适合专业环境的行为

## 执行职责

社区维护者负责澄清和执行我们可接受的行为标准，并将对其认为不当、威胁、冒犯或有害的任何行为采取适当且公平的纠正措施。

## 适用范围

本行为准则适用于所有社区空间，并且在个人在公共空间正式代表社区时也适用。

## 执行

辱骂、骚扰或其他不可接受的行为可通过 [conduct@calnexus.dev](mailto:conduct@calnexus.dev) 向负责维护的项目维护者举报。所有投诉都将得到及时和公正的审查和调查。

所有社区维护者都有义务尊重举报者的隐私和安全。

## 执行方针

项目维护者将遵循以下社区影响方针来判断违反本行为准则行为的后果：

1. **更正**：社区影响不严重且明确愿意遵守准则的违规行为，可通过私下警告明确违规性质并说明为何行为不当。
2. **警告**：针对社区维护者认为持续的一系列违规行为，将发出警告，并可能伴随一段时间内的后续行为后果。在指定时间内不得与相关人员互动，包括主动与非主动。这包括在社区空间及外部渠道中的互动。违反这些条款可能导致临时或永久封禁。
3. **临时封禁**：针对严重的社区影响行为，包括持续的不当行为，将进行指定时间内的封禁。在封禁期间，不得与相关人员进行公开或私下互动，包括主动与非主动。违反这些条款可能导致永久封禁。
4. **永久封禁**：针对持续社区影响的行为模式，包括对个人的骚扰、侮辱或贬损，或对某些群体的仇恨行为，将进行永久封禁。
